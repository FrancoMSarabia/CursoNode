para instalar modulos de node: npm install

para ejecutar el programa: node app.js

para ejecutar el programa y refrescar al momento de hacer cambios: nodemon app.js

para ejecutar un comando especificado en el "package.json" en este caso el comando especial start: npm start

para ejecutar un comando especificado en el "package.json" que no es especial: npm run nombre_comando



Comandos para heroku:

heroku create: Es para crear el proyecto en el heroku

git push heroku master: Es para desplegar la app a heroku

herohu open: Es para abrir el proyecto
